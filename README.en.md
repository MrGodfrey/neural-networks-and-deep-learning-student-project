# Neural Networks and Deep Learning Student Project Repository

> This repository is the companion student project repository for the course Neural Networks and Deep Learning.
>
> It collects student-contributed course materials, code, and experimental content. Strong student contributions may be curated here as course-related reference material, and students are encouraged to contribute actively.

## What Students Can Contribute

Students are welcome to contribute through Pull Requests, including but not limited to:

1. Slides they created
2. Interesting programming projects, especially machine-learning-related projects
3. Jupyter Notebooks suitable for labs, classroom demos, or self-study
4. Other high-quality course-related materials

## Pull Request Contribution Guide

Please contribute by submitting a Pull Request.

1. Fork this repository to your own GitHub account.
2. Create a clearly named branch in your fork.
3. Add your materials and include a short description when helpful so that I can understand the purpose and context.
4. Commit your changes and push them to your fork.
5. Open a Pull Request to this repository and explain what you are contributing, where it may be useful, and anything I should pay attention to during review.
6. I will review the Pull Request and decide whether to merge it into the course project repository.
